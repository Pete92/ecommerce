BackEnd REST API


##VERKKOKAUPAN BackEndin toiminta käyttäen Postmania##

Palauttaa kaikki tuoteet (GET)
http://localhost/REST_API/items/single_read.php  

Palauttaa halutun id:en tiedot(GET)
http://localhost/REST_API/items/single_read.php/?id=3 


Uusi tallennus (POST)
Lähetetään Postmanilla POST pyyntö täyttämällä vaadittavat inputit
{
    "title": "...",

}
http://localhost/REST_API/items/create.php



Tietyn tuotteen päivitus(POST)
Lähetetään Postmanilla POST pyyntö täyttämällä vaadittavat inputit
{
    "id": 12,
    "title": "...",

}
http://localhost/REST_API/items/update.php


Tietyn tuotteen poisto(DELETE)
Lähetetään Postmanilla DELETE pyyntö täyttämällä vaadittavat inputit
{
    "id": 12
}
http://localhost/REST_API/items/delete.php




##Rekisteröityminen ja Kirjautuminen käyttäen Postmania##

Rekisteröityminen
Lähetetään Postmanilla POST pyyntö täyttämällä vaadittavat inputit
{
    "name": "testi",
    "email": "testi@testi.fi",
    "password": "testi123"
}
http://localhost/REST_API/user/register.php


Kirjautuminen
Lähetetään Postmanilla POST pyyntö täyttämällä vaadittavat inputit
{
    "email": "testi@testi.fi",
    "password": "testi123"
}
http://localhost/REST_API/user/login.php

Haetaan halutun käyttäjän tiedot (POST)
Kun kirjaudutaan, responsessa on token ja tätä tokenia käyttämällä saadaan käyttäjän tiedot

Key             Value

Authorization   Bearer <token>

http://localhost/REST_API/user/user-info.php
